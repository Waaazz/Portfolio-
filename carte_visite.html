Générateur de Carte de Visite Numérique

Projet Freelance – Janvier 2025

Description du projet

Plateforme web permettant aux professionnels de créer facilement leur carte de visite numérique personnalisée. L'application génère un QR code dynamique pour un partage rapide et interactif.

Fonctionnalités

  • Création de carte de visite personnalisée avec informations professionnelles
  • Génération automatique de QR code unique
  • Interface moderne et responsive
  • Stockage des données en temps réel
  • Partage facile via lien ou QR code

Technologies utilisées

  • Frontend : React.js
  • Backend/Base de données : Firebase (Firestore, Authentication)
  • Hébergement : Render
  • Autres : QR Code Generator API
