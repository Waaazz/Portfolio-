LocaBien

Plateforme de Location de Biens Immobiliers au Burkina Faso

Description du projet

LocaBien est une plateforme web innovante qui facilite la location de biens immobiliers au Burkina Faso. Elle permet aux utilisateurs de rechercher, consulter et réserver des biens (appartements, maisons, villas, studios, bureaux, terrains) de manière simple et intuitive.

Fonctionnalités

  • Recherche et filtrage de biens par catégories (Appartement, Maison, Villa, Studio, Bureau, Terrain)
  • Système d'authentification et inscription des utilisateurs
  • Réservation en ligne des propriétés
  • Interface d'administration pour la gestion des biens
  • Upload et gestion d'images via Cloudinary
  • Affichage des prix en FCFA avec disponibilité

Technologies utilisées

  • Backend : Python / Django
  • Frontend : HTML5, CSS3, JavaScript, Bootstrap
  • Base de données : PostgreSQL
  • Stockage images : Cloudinary
  • Hébergement : Render
