Apollon+ - Chatbot Paris Sportifs

Projet Freelance – 2024

Description du projet

Apollon+ est un chatbot conversationnel conçu pour assister les utilisateurs dans le domaine des paris sportifs en ligne. Il offre une interface intuitive permettant d'obtenir rapidement des informations sur les plateformes de paris.

Fonctionnalités

  • Assistance pour la création de compte sur les plateformes de paris
  • Informations sur les codes promotionnels disponibles
  • Support client interactif
  • Navigation guidée vers les plateformes partenaires
  • Interface conversationnelle avec menu interactif

Technologies utilisées

  • Frontend : HTML5, CSS3, JavaScript
  • Architecture : Application web statique
  • Hébergement : GitHub Pages
  • UX : Interface conversationnelle avec emojis
